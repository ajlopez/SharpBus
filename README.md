# SharpBus

Simple bus implementation in C#, inspired in Mule ideas

Work in Progress

## Design Notes

- Configuration by code, first
- Operations on Payload
- Operations on Message

## Development

```
git clone git://github.com/ajlopez/SharpBus.git
cd SharpBus
```

Open `Src\SharpBus.sln` in Visual Studio
